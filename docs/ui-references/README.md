# Clay UI References

Эта папка хранит распакованные UI reference exports от Gemini рядом с рабочим проектом `Clay`.

## Текущий Эталон

- `clay-mission-control-ui-v17/` - текущий visual reference для `v17` UI-pass.

## Как Использовать

- Живое приложение остается в `frontend/`.
- Reference exports не копируются целиком в `frontend/`.
- Из reference exports переносим только проверенные layout, visual language и UX-решения, адаптируя их к live hooks/API.
- Zip-архивы остаются в Obsidian как cold archive.

## Когда Появится V18

Лучше сначала добавить новый export рядом:

`docs/ui-references/clay-mission-control-ui-v18/`

После визуального сравнения можно либо оставить оба reference, либо удалить устаревший локальный export отдельным осознанным коммитом.
